如何入门？

  • 按照 install-LaTeX 安装 TeXLive 2019

    • 找到最新的 releases，点击下载 Install-LaTeX.pdf

  • 按照 lshort-zh-cn 入门

    • 上同，由于国内访问 GitHub 网速会较慢，可以找其他方式下载，最新版为 6.01

  • 部分编辑器配置

    • TeXworks
    • TeXstudio
    • VsCode
    • Sublime Text

你用 CTeX 套装的理由是？

  • 写文章 / 写东西 / 排版兴趣爱好 / 投稿国外期刊

    • 这不是用CTeX套装的理由，请使用 TeXLive

  • 投稿中文核心期刊

    • 用吧

  • 打比赛，数学建模等

    • 如果你们的老师给了你们所谓的（带有 cct、cjk 字样的）模板 / 要求使用 CTeX 套装，走好不送
    • 如果没有老师要求，请使用 TeXLive
