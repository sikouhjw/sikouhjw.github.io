  • [LaTeX 发行版] 2019 年，不用 CTEX 套装的新理由

如何入门？

  • 按照 install-LaTeX 安装 TeXLive 2019
    • 找到最新的 releases，点击下载 Install-LaTeX.pdf
  • 按照 lshort-zh-cn 入门
    • 安装了 TeXLive 2019 后，命令行执行 texdoc Lshort-zh （避免把 l 认作 i，用了大写）即可打开
    • 从网络下载，由于国内访问 GitHub 网速会较慢，可以找其他方式下载，最新版为 6.01
    • 为方便下载，上传到了蓝奏云
  • 其它扩展阅读
    • 一份其实很短的 LaTeX 入门文档
    • 回「LaTeX 的罪与罚」#入门教程
    • LaTeX\LaTeXLATE​X 入门，《书香中国》在线阅读，建议还是买纸质书
  • 部分编辑器配置教程
    • TeXworks
    • TeXstudio
    • VsCode
    • Sublime Text

你用 CTeX 套装的理由是？

  • 写文章 / 写东西 / 排版兴趣爱好 / 投稿国外期刊

    • 这不是用 CTeX 套装的理由，请使用 TeXLive

  • 投稿中文核心期刊

    • 用吧

  • 打比赛，数学建模等

    • 如果你们的老师给了你们所谓的（带有 cct、cjk 字样的）模板 / 要求使用 CTeX 套装，走好不送
    • 如果没有老师要求，请使用 TeXLive
