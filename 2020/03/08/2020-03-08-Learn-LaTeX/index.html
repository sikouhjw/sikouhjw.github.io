劝退 CTeX 套装用户

首先劝退 CTeX 套装用户，如果想正经的入门 LaTeX\LaTeXLATE​X，请卸载 CTeX 套装，并备份环境变量。

你用 CTeX 套装的理由是？

  • 写文章 / 写东西 / 排版兴趣爱好 / 投稿国外期刊

    • 这不是用 CTeX 套装的理由，请使用 TeX\TeXTE​X Live

  • 投稿中文核心期刊

    • 用吧

  • 打比赛，数学建模等

    • 如果你们的老师给了你们所谓的（带有 cct、cjk 字样的）模板 / 要求使用 CTeX 套装，走好不送，后果自负，报错了请找自己的老师
    • 如果没有老师要求，请使用 TeX\TeXTE​X Live，欢迎使用全国大学生数学建模竞赛 LaTeX\LaTeXLATE​X 模板：JXUSTmodeling

为什么不推荐使用 CTeX 套装了

  • 《[LaTeX 发行版] 2018年，为什么不推荐使用 CTeX 套装了》
  • 《[LaTeX 期刊模板] els-cas-templates 不支持 CTEX 套装》
  • 《[LaTeX 期刊模板] siamart 不支持 CTEX 套装》

学习 LaTeX 的（一般）流程

如何入门？

  • 按照 install-LaTeX 安装 TeX\TeXTE​X Live 2020
    • 找到最新的 releases，点击下载 Install-LaTeX.pdf
  • 按照 lshort-zh-cn 入门，打开方式：
    • 完整安装 TeX\TeXTE​X Live 2020 后，命令行执行 texdoc Lshort-zh-cn （避免把 l 认作 i，用了大写）即可打开
    • 网络下载：
      • GitHub
      • CTAN
      • TeXdoc
  • 其它扩展阅读
    • 一份其实很短的 LaTeX 入门文档
    • 回「LaTeX 的罪与罚」#入门教程
  • 部分编辑器配置教程
    • TeXworks
    • TeXstudio
    • VsCode
    • Sublime Text

  • 偏使用

  • 偏底层 / 开发
