美赛模板成功编译步骤

也仅是编译通过，美赛视频可以看 lvqaq-美赛学习

  1. 下载 美国数学建模 2020 年更新版本 6.3 的压缩包

  2. 解压到一个文件夹中

  3. 在工作路径下打开 cmd.exe（也就是命令行），输入 xelatex mcmthesis-demo.tex，回车

    • 成功编译，结束。

    • 没成功编译，检查发行版是否为 TeXLive 2019

      • 若是，将 log 文件发去 LaTeX 工作室问答网站

      • 若不是，请卸载干净再按照 安装 TeXLive

真心劝退美赛用 LaTeX\LaTeXLATE​X，内容比形式重要。还有如果你觉得模板哪里的设置（可能）不对，也不要用模板，用 word，word 好

美赛 LaTeX\LaTeXLATE​X 基本常识

  • 首段不缩进是正常的
  • 模板是符合比赛要求的
  • 摘要页和标题页，说明手册有说怎么去除
  • 劝你用 CTeX 套装编译的不是蠢就是坏
  • LaTeX\LaTeXLATE​X 里的图表（实质是指浮动体）飘来飘去（固定不了位置）是正常的，视频参考【LaTeX 从 install 到 uninst】第八期：图片、盒子与浮动体，文章参考LaTeX 中的浮动体：基础篇

修正美国数学建模 2020 年更新版本 6.3 的一个 bug

由于 \section 和 \subsection 后首段缩进了，可以将 cls 文件中第 298、302 行的 -0pt 改为 -1sp，问题解决
