【记录贴】在树莓派中安装 TeX Live
2020-08-07| LaTeX|字数总计: 1.8k|阅读时长: 9 分钟

前言

为什么要在树莓派上安装 TeX\TeXTE​X Live 呢？因为好玩可以学到 Linux 的一些东西，还可以拿来装逼体验 Linux 下跑 LaTeX\LaTeXLATE​X 的速度。

以此贴笼统的记录全过程，配置树莓派的过程比较简略，毕竟详细的步骤网上都有很多，搜一搜就能找到。

正文

用到的主要设备：

  • Raspberry Pi 4 Model B
  • 网线
  • 读卡器
  • tf 卡
  • Win 10 笔记本电脑

购买

找到官网推荐的购买链接，选择

  • Select a model: 4GB RAM
  • Country: 中国

然后找一家淘宝店购买。（注意 RPi 4B 的发热比较严重，建议购买主动散热装置）

组装及烧录

组装看配套的说明手册。

烧录我是用了淘宝店提供的百度网盘里的软件。（也可以用官网提供的 Downloads，总之不是重点）

格式化 tf 卡、烧录好系统之后，在 tf 卡的主目录下创建一个

  • 文件名为 ssh
  • 不带后缀

的文件，代表开启树莓派的 SSH 功能。

用 SSH、VNC 连接树莓派

参考知乎问题《没有显示器可以使用树莓派么，初次安装过程中呢？》。（温馨提示：可以连接上树莓派的 IP 地址可能要等一段时间才能刷出来，如果测试了好几个 IP 都不行，就多等会）

安装 TeX Live

因此采用联网安装的方式。

开始安装

然后根据自己的需求选择性安装，由于树莓派我是打算拿来跑程序而不是拿来查文档的，因此我选择了不安装说明文档。安装界面如下：
