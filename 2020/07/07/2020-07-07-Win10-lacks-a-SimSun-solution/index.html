Win 10 缺少 SimSun 时的解决方案
2020-07-07| LaTeX|字数总计: 409|阅读时长: 1 分钟

前言

操作系统一般分为：

  • Windows
  • macOS
  • Linux

目前 ctex 宏集的字体设置会区分这四种操作系统

  • macOS Old
  • macOS New
  • Windows
  • 其他

（在 XeLaTeX 下）对应调用的字库也是不同的，其中 Windows 会调用中易字库 + 微软雅黑，本文对 Windows 缺中易字库的某些字体的情况作出详细的解决方法。

正文

临时解决方案

调用 TeX\TeXTE​X Live 自带的开源中文字体——Fandol 字库。

优点：安装了完整的 TeX\TeXTE​X Live 就能用。

缺点：

  • 缺字较严重
  • 部分字设计崩了

治根的解决方案

缺字安字

用搜索引擎搜索「中易 xxx 体 下载」，其中 xxx 为宋体、黑体、楷体……

下载后，全选右键，为所有用户安装，即可。

使用思源字体

介绍链接：

  • 思源宋体
  • 思源黑体

下载指南：

  • 思源宋体
  • 思源黑体

提供两个 GitHub 加速下载工具：

  • https://d.serctl.com/
  • https://toolwa.com/github/

下载类型为 Language-specific OTFs 的思源宋体/黑体，为所有用户安装，测试以下代码

GBZenKai-Medium 是文鼎P L简中楷，用来作 mainfont 的 \itshape。TeX\TeXTE​X Live 自带了文鼎P L简报宋（BousungEG-Light-GB）和文鼎P L简中楷（GBZenKai-Medium），可以拿来用（不如 Fandol 多，Fandol 有宋体、楷体、黑体、仿宋，其中宋体、黑体还有对应的粗体）。
